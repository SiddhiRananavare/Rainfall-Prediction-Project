# Rainfall-Prediction-Project
Developed a rainfall prediction model using historical weather data. Performed data preprocessing, handled missing values, and conducted EDA to identify patterns. Applied machine learning algorithms to forecast rainfall and evaluated performance using accuracy and other metrics for better decision-making.
